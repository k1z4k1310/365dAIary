よくあるエラー
ERROR
APIキーを設定してください
›
1365dAIary本体を開いて ⚙ ボタンをタップ
2APIキー欄に取得したキーを貼り付ける
3モデルとキーのプロバイダーを合わせる（OpenAIモデル→OpenAIキー）
取得先: OpenAI / Anthropic / Google
ERROR
通信エラー: Load failed
›
SafariがAPIをブロックしている可能性が高い
1iPhoneの設定 → Safari → プライバシーとセキュリティ
2クロスサイトトラッキングを防ぐ をOFF
3それでも出る場合: 設定 → Apple ID → iCloud → プライベートリレー OFF
ERROR
401 Unauthorized（認証エラー）
›
1設定 ⚙ でAPIキーをコピペし直す（スペースに注意）
2モデルとキーのプロバイダーが一致しているか確認
WARN
429 Rate limit（送信しすぎ）
›
11〜2分待って再送信
2より軽いモデル（gpt-4o-mini / Haiku / Gemini Flash）に変える
WARN
タイムアウト / 応答が遅い
›
1軽いモデルに変える
2新しいセッションを始める
3Wi-Fiに切り替える
INFO
コスト上限に達して送信できない
›
1設定 ⚙ → コスト上限を引き上げるか 0（無制限）に変更

›

›
💬普段使い → GPT-4o-mini / Claude Haiku / Gemini Flash

FAQ
APIキーは安全？
›
APIキーはブラウザ（localStorage）にのみ保存。サーバーには送信されません。
